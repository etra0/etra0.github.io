Datacampfire Datathon

2018-06-13

Datacampfire hace poco hizo una Hackathon para crear visualizaciones sobre los gastos fiscales, otorgando los datos de éstos (ergo, Datathon). Junto a unos amigos, decidimos participar en ésta, ocupando nuestra pequeña disponibilidad de tiempo para construir una plataforma con el fin de utilizar herramientas nuevas tales como Altair + vega-lite.

El equipo se llamaba DMCTeam, y estaba compuesto por dos Data Scientist y yo, un informático. Fue un entretenido trabajo, pero todos quedamos con ganas de haberle dedicado muchísimo más tiempo.

Principalmente, quería darle un formato de revista, creando un layout bastante interesante que me terminó gustando mucho. No tengo idea de por qué usé Angular para la construcción, pero sirvió, y… if it works, it ain’t stupid.
